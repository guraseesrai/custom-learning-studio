# 🎓 Custom Learning Studio

**Custom Learning Studio** is an interactive, AI-powered tool that generates structured learning scripts from PDF or Word documents based on user-defined learning outcomes. Built with **FastAPI**, **LangChain**, and **OpenAI GPT-4**, the app ships with a modern **Bootstrap 5** frontend featuring dark mode, confetti animation, live stats, and instant JSON downloads.

---

## ✨ Key Features

- 🤖 **AI-Generated Content**: Powered by GPT-4 via LangChain for intelligent script generation
- 📄 **Multi-Format Support**: Upload `.pdf` or `.docx` files to extract book/course content
- 🎯 **Custom Learning Outcomes**: Enter specific learning goals and receive tailored structured scripts
- 💾 **Instant Downloads**: Download results immediately as formatted JSON files
- 🎉 **Modern UI**: Animated interface with confetti effects, dark-mode toggle, and live counters
- ☁️ **Cloud Backup**: Optional AWS S3 integration for script storage
- 📊 **Real-time Stats**: Live processing statistics and progress tracking
- 🎨 **Responsive Design**: Bootstrap 5 ensures perfect display on all devices

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- OpenAI API key
- Git (for cloning)

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/custom_learning_studio.git
   cd custom_learning_studio
   ```

2. **Create virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Set up environment variables**
   
   Create a `.env` file in the project root:
   ```ini
   OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   AWS_ACCESS_KEY_ID=your_aws_access_key (optional)
   AWS_SECRET_ACCESS_KEY=your_aws_secret_key (optional)
   AWS_DEFAULT_REGION=us-west-2 (optional)
   ```

5. **Run the FastAPI server**
   ```bash
   uvicorn main:app --reload
   ```

6. **Access the application**
   
   Open your browser and navigate to `http://localhost:8000`

## 📖 Usage Guide

### Basic Workflow

1. **Upload Document**: Select and upload a PDF or Word document containing your source material
2. **Define Learning Outcomes**: Enter specific learning objectives you want to achieve
3. **Generate Script**: Click "Generate Learning Script" to process your content with AI
4. **Review Results**: Examine the structured learning script generated by GPT-4
5. **Download**: Save your results as a JSON file for future reference

### Supported File Formats

- **PDF Files** (`.pdf`): Text extraction using PyPDF2 or similar libraries
- **Word Documents** (`.docx`): Content parsing using python-docx

### Learning Outcome Examples

- "Understand the fundamental principles of machine learning"
- "Learn to implement basic data structures in Python"
- "Master advanced SQL query optimization techniques"
- "Develop proficiency in React component architecture"

## ⚙️ Configuration

### Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `OPENAI_API_KEY` | Your OpenAI API key for GPT-4 access | Yes |
| `AWS_ACCESS_KEY_ID` | AWS access key for S3 backup | No |
| `AWS_SECRET_ACCESS_KEY` | AWS secret key for S3 backup | No |
| `AWS_DEFAULT_REGION` | AWS region for S3 bucket | No |

### API Settings

The application uses OpenAI's GPT-4 model with optimized prompts for educational content generation. You can modify the LLM parameters in `llm_service.py`:

```python
# Example configuration
temperature = 0.7
max_tokens = 2000
model = "gpt-4"
```

## ☁️ AWS S3 Integration (Optional)

To enable cloud backup functionality:

1. **Configure AWS credentials** in your `.env` file
2. **Create an S3 bucket** for storing generated scripts
3. **Update bucket name** in `book_processor.py`:

```python
def backup_to_s3(file_path, bucket_name="your-bucket-name"):
    upload_file_to_s3(file_path, bucket_name, folder="learning_scripts")
```

### S3 File Structure
```text
your-bucket-name/
└── learning_scripts/
    ├── script_2024_01_15_001.json
    ├── script_2024_01_15_002.json
    └── ...
```

## 📁 API Endpoints

### Core Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/` | Main application interface |
| `POST` | `/upload` | Upload and process document |
| `POST` | `/generate` | Generate learning script from content |
| `GET` | `/download/{filename}` | Download generated JSON file |
| `GET` | `/health` | API health check |

### Request/Response Examples

**Upload Document:**
```bash
curl -X POST "http://localhost:8000/upload" \
     -H "accept: application/json" \
     -H "Content-Type: multipart/form-data" \
     -F "file=@document.pdf"
```

**Generate Script:**
```bash
curl -X POST "http://localhost:8000/generate" \
     -H "accept: application/json" \
     -H "Content-Type: application/json" \
     -d '{"learning_outcomes": "Master Python fundamentals", "content": "extracted_text"}'
```
